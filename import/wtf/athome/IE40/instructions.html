Microsoft® Internet Explorer 4.0 Provided by Compaq
Download Instructions
    It's easy! Just follow these simple, step-by-step instructions to download and install Internet Explorer 4.0
 


Note: We recommend printing these instructions for future reference before installing Internet Explorer 4.0. And be sure to review the Installation notes before you start.

  1. Click on the Download button to the left to run setup.
  2. In the first dialog box, choose either Save As to save the setup files on your hard drive or Open File to open the setup files directly.
    • If you choose Open File, Internet Explorer setup begins automatically, and you remain connected to the Internet while the setup program downloads the necessary files, then installs and configures Internet Explorer.
    • If you chose Save As, select the location on your hard disk where you want to save the Internet Explorer setup files. After the setup files have been saved to your hard drive, and while you're still connected to the Internet, run setup by double-clicking on the ie4setup.exe file in Windows Explorer.
  3. In the first setup dialog box, select which package of Internet Explorer 4.0 Suite components you want to download. You can choose from the following Internet Explorer 4.0 Suite component packages:
    • Minimal: Includes Internet Explorer 4.0, Java Support, and Microsoft Internet Connection Wizard.
    • Standard: To the Minimal installation we add Microsoft OutLook Express and Microsoft Wallet.
    • Full: To the Standard installation we add NetMeeting, NetShow, FrontPage Express, Microsoft Web Publishing Wizard, and Microsoft Chat 2.0.

    Note: All of these packages include the Compaq customizations described previously. Be sure to allow enough disk space for your chosen package; see the system requirements.

  4. Restart your computer after the setup program ends.

    Important: In the event that your computer doesn't restart properly after the Internet Explorer 4.0 installation program prompts you to shut down, you should turn your computer's power switch OFF. Wait 10 seconds, then turn your computer on to complete your installation successfully.

Installation notes

  • If your computer stops responding during installation, restart the computer. Internet Explorer 4.0 will complete the installation when your computer restarts.
  • If you have installed a version of Internet Explorer 4.0 previously and are not using a Web page as your background, the Presario Planet Active Desktop wallpaper won't be set up during this installation. After the installation ends, click on Start. Point to Settings, then to Active Desktop, then to Customize my Desktop. On the Background tab, select the Oct97 file from the list of wallpapers. Click on OK to set up the Presario Planet wallpaper.
  • Some channels and newsgroups may be inappropriate for children. Be sure to supervise your children when they're surfing the Web.
