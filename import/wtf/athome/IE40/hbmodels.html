Microsoft® Internet Explorer 4.0 Provided by Compaq
Important Notes for Home Base Users
    Some Compaq Presario computers offer users a program called Home Base that, like Internet Explorer 4.0, is designed to make the computer more customizable for individual users. Home Base users should read this page for special information before downloading Internet Explorer 4.0.
 


Internet Explorer 4.0 offers customization of the desktop interface through the Active Desktop. When it is installed on a Compaq Presario with Home Base, there is an initial conflict.

To remove this conflict, Compaq has provided a program that disables the Home Base shell. If your computer appears on this list, you should run this Prepare Your Computer program before installing Internet Explorer 4.0. If you don't run the Prepare Your Computer program before you install Internet Explorer 4.0, your computer may not function properly.

If you're using one of the computers listed and run the Prepare Your Computer program, the program makes these changes to your computer:
  • Home Base is removed from your computer. Pressing the SmartQ button no longer opens Home Base, and Home Base doesn't start up when you start your computer.
  • You can't switch among multiple users through Home Base. However, you can switch among users by using the Windows login feature. To change users, click on Start, then on Log off.... Windows closes all open programs and prompts you to log in.
  • You can't open the Help Center through Home Base. To open the Help Center, click on Start, then on Run, then enter the following line in the Run dialog box:
    c:\compaq\help\cpqhelp.exe
Important: If you don't want Home Base removed from your computer, you should wait for the next release of Microsoft Internet Explorer provided by Compaq. This version, which will be available soon, will correct the conflict with Home Base, and you'll be able to install Internet Explorer and keep Home Base on your computer.

To download Internet Explorer 4.0 now, follow these instructions.
  1. Click on the Prepare Your Computer button below to remove Home Base from your computer. In the dialog box that appears, be sure to choose the Open option to run the program from the Web.

  2. After you've prepared your computer, proceed with the download by clicking on the Download button below.

    Important: In the event that your computer doesn't restart properly after the Internet Explorer 4.0 installation program prompts you to shut down, you should turn your computer's power switch OFF. Wait 10 seconds, then turn your computer on to complete your installation successfully.
