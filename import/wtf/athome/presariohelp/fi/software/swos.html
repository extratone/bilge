Mikä on käyttöjärjestelmä?

Käyttöjärjestelmä on ohjelmisto, joka ohjaa tietokonetta ja siihen kytkettyjä oheislaitteita. Käyttöjärjestelmä kertoo tietokoneelle, kuinka ohjelmia käytetään ja se hallitsee tietokoneeseen asennettua laitteistoa, sekä sisäistä että ulkoista.

Käyttöjärjestelmä voi olla merkkipohjainen tai graafinen. Merkkipohjainen käyttöjärjestelmä kuten MS-DOS sallii komentojen kirjoittamisen kehotteeseen tietokoneen ohjaamiseksi. Graafinen käyttöliittymä (Graphical User Interface, GUI) sallii komentojen lähettämisen tietokoneelle napsauttamalla kuvaketta tai valitsemalla nimikkeet valikoista. Windows 95:ssä on graafinen käyttöliittymä.

Useimmat käyttöjärjestelmät voivat käsitellä joko 16-bittisiä tai 32-bittisiä ohjelmia tai molempia. Microsoft Windows 3.x voi ajaa vain 16-bittisiä ohjelmia; Microsoft Windows 95 voi ajaa sekä 16-bittisiä että 32-bittisiä ohjelmia.

Presario-tietokoneet tukevat Windows 95 -äyttöjärjestelmää.
